# Radar-7674761

Project created to report bug on UISearchBar for tvOS

# Description
The prompt label of UISearchBar is so tiny in tvOS that it is imposible to read it from the distance that the device is usually used.

![](preview.png)

# Expected 
The prompt label is readable from a reasonable distance.

# Steps
A project reproducing the problem can be found in this repo.

# Environment

- Xcode Version 11.3.1 (11C505)
- tvOS 13.4 (17L256)
